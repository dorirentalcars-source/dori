# Dori Rental Cars

Static website ready for Netlify deployment.

## Includes
- Luxury dark landing page
- Fleet section with search/filter
- Reservation form configured for Netlify Forms
- Newsletter form configured for Netlify Forms
- Contact, reviews, about, footer, legal pages
- Optimized car images

## Netlify Forms
The forms are already set up with:
- `data-netlify="true"`
- hidden `form-name` fields
- success redirect to `thank-you.html`

To receive reservation notifications at `dorirentalcars@gmail.com`, set the form notification email inside the Netlify dashboard after deployment.

## Deploy
Upload the folder to Netlify or drag-and-drop the zipped project.
